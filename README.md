AI-Driven Hospital Bed & Resource Allocation Optimizer
Track: Healthcare | Societal
Problem Statement:
Hospitals face daily challenges in dynamically allocating beds, ICU units, and medical staff based on
patient inflow. Build an AI forecasting and recommendation system that predicts patient admission
volumes and suggests optimal resource allocation shifts — reducing bottlenecks and improving
patient throughput.

Inputs:
Historical patient admission and discharge records (open health datasets or simulated)
Real-time bed occupancy data (simulated stream)
Staff roster and shift schedules

Outputs:
24-hour and 7-day patient admission forecasts
Bed allocation recommendations by ward
Staff reallocation suggestions per shift
Operations dashboard with live occupancy and forecast overlays

Evaluation Metrics:
Admission forecast MAPE (target: < 10%)
Allocation recommendation acceptance rate (human-judged)
Dashboard real-time update latency (target: < 2 seconds)
